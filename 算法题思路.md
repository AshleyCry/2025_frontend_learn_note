### 随机链表的复制

（方法1）

copyNode：

​	如果map中有node对应的新节点，直接返回这个节点

​	如果没有，新建节点newNode,newNode放入map

​	newNode.val = node.val

​	newNode.next = node.next ? copyNode(node.next) : null

​	newNode.random= node.random? copyNode(node.random) : null

​	return newNode

维护dummy,pointer指向dummy

头节点不空的时候：

​	pointer.next = copyNode(head)

​	pointer = pointer.next
